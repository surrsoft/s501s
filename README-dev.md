# s501s

## описание

- расширение для VS Code: аналог расширения VS Code Pets. Тоже пиксельная графика, только сюжетом является вид из окна движущегося поезда. Отображается пейзаж, мелькают деревья, столбы, электрические провода, поля, города, станции, животные и прочее. Разные биомы, разное время суток, разная погода.
